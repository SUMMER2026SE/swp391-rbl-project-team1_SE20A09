# Skills System

## Danh sách Skills khả dụng

Mỗi skill chứa kiến thức chuyên sâu, patterns, và best practices.
Khi gặp task liên quan, hãy đọc skill tương ứng trước khi code.

## Backend Skills

| Skill | Khi nào dùng |
|---|---|
| `spring-boot/` | Tạo endpoint, config beans, profiles, application.yml |
| `java-patterns/` | Design patterns, Java 21 features, clean code |
| `jwt-auth/` | Implement JWT filter, refresh token, SecurityConfig |
| `postgresql/` | Query optimization, Flyway migrations, indexes |
| `security-auditor/` | Security review, OWASP, input validation |

## Frontend Skills

| Skill | Khi nào dùng |
|---|---|
| `nextjs-best-practices/` | App Router, Server Components, routing |
| `react-patterns/` | Custom hooks, composition, performance |
| `typescript-pro/` | Type safety, generics, utility types |
| `tailwind-patterns/` | Responsive design, dark mode, utilities |
| `zustand-store-ts/` | Global state management, store setup |
| `shadcn/` | Component installation, customization |
| `frontend-design/` | UX patterns, animation, premium feel |

## Infrastructure Skills

| Skill | Khi nào dùng |
|---|---|
| `docker-expert/` | Dockerfile, compose, networking |
| `github-actions-templates/` | CI/CD pipeline, workflows |
| `powershell-windows/` | Windows scripts, automation |

## Debug & Review Skills

| Skill | Khi nào dùng |
|---|---|
| `systematic-debugging/` | Root cause analysis, stack traces |
| `security-auditor/` | Security vulnerabilities review |

## Cách dùng

Khi task liên quan đến skill nào, AI sẽ tự động apply knowledge từ file skill tương ứng.
Có thể request cụ thể: *"Dùng spring-boot skill để implement endpoint này"*
