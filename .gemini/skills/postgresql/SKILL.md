## Useful Queries

```sql
-- Xem tất cả bảng
SELECT tablename FROM pg_tables WHERE schemaname = 'public';

-- Xem cấu trúc bảng
\d stadiums

-- Xem Flyway history
SELECT * FROM flyway_schema_history ORDER BY installed_rank;

-- Check indexes
SELECT indexname, indexdef FROM pg_indexes WHERE tablename = 'bookings';

-- Thống kê bookings theo tháng
SELECT DATE_TRUNC('month', booking_date) as month,
       COUNT(*) as total,
       SUM(total_price) as revenue
FROM bookings
WHERE booking_status = 'Completed'
GROUP BY 1 ORDER BY 1;
```

## Index Strategy

```sql
-- Đã có trong V1 (indexes section)
-- Thêm nếu cần cho các query phức tạp:

-- Full-text search stadium name
CREATE INDEX idx_stadiums_name_fts
ON stadiums USING gin(to_tsvector('english', stadium_name));

-- Booking date range queries
CREATE INDEX idx_bookings_date
ON bookings(booking_date DESC);

-- Time slot availability
CREATE INDEX idx_time_slots_start_end
ON time_slots(stadium_id, start_time, end_time)
WHERE slot_status = 'Available';
```

## Common Gotchas

- **Port 5432 bị chiếm** bởi PostgreSQL Windows local → dùng 5433
- **`SERIAL` vs `IDENTITY`**: Dự án dùng `SERIAL` (tương đương `INT GENERATED BY DEFAULT AS IDENTITY`)
- **Timestamps**: Dùng `TIMESTAMP` (không có timezone) — đồng nhất với SQL Server design gốc
- **Enum-like columns**: Dùng `VARCHAR + CHECK constraint` thay vì PostgreSQL enum type (dễ alter hơn)
